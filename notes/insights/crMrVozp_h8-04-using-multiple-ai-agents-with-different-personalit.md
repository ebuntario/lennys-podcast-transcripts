Different [[concepts/ai-agents|AI agents]] have distinct styles and strengths, much like human team members. Using a suite of agents (e.g., Claude Code, Friday, Charlie) for different tasks—like code review, brainstorming, or analysis—provides varied perspectives and outputs. This approach mirrors building a "team of Avengers" where each agent excels in a specific area, increasing overall [[concepts/team-leverage|team leverage]] and output quality.
