NLX (Natural Language Interface) represents the next evolution of user experience, moving beyond rigid graphical interfaces to more elastic conversational ones. This doesn't mean design is eliminated; conversations have their own grammars, structures, and invisible UI elements that must be intentionally crafted. New constructs like prompts, editable plans, and progress indicators are emerging as fundamental [[concepts/ai-interaction-design|interaction design]] elements for AI products.
