An effort pact is a precommitment that makes getting distracted more difficult, acting as a last line of defense. Examples include using an outlet timer to shut off your internet router at a set time or using the Forest app that kills a virtual tree if you use your phone. These [[concepts/precommitment|precommitments]] insert mindfulness into previously mindless behaviors.
