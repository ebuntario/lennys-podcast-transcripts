Create a simple system: draft a template email requesting a 10-minute phone chat, hypothesize the user type you need to hear from, and send a few requests each week. Unstructured conversations provide richer insights than surveys. This consistent [[concepts/customer-development|customer development]] habit builds a superior mental model of user needs and problems.
