A canonical document serves as the single source of truth for any project, linking to all other relevant materials. This eliminates confusion when ramping up, as everyone knows exactly where to find essential information like work streams, owners, and processes. Establishing [[concepts/project-management|project management]] clarity through a central doc prevents wasted time from multiple conflicting answers.
