Adopting an [[concepts/open-source|open source model]] reduces friction for users to get started, which fuels organic sharing and adoption across diverse companies and industries. This large user base then attracts partners to build integrated solutions, creating a powerful network effect that benefits everyone on the standard and spins the growth flywheel faster.
