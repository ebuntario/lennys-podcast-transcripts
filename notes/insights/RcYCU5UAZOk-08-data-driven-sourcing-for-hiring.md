Use external data to identify the best companies and people to hire for a specific role. For example, to find great email marketing talent, analyze which respected companies drive the highest percentage of web traffic via email using tools like SimilarWeb. Then source candidates from those top-performing teams. This [[concepts/data-driven-hiring|data-driven approach]] complements traditional network-based searches and uncovers under-the-radar talent pools.
