When acquiring products with strong user attachment, the integration process must preserve the core experience that created the emotional connection. This requires careful [[concepts/product-migration|product migration]] planning that maintains user data like stats and streaks while enabling broader platform integration. Transparency about technical constraints during migration helps maintain user trust when unexpected issues arise.
