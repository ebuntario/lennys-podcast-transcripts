Engineering teams at Block that are very AI-forward report saving 8-10 hours per week using tools like their internal agent Goose. This self-reported metric is considered the worst it will ever be, establishing a baseline for future [[concepts/ai-productivity|AI productivity]] gains. The value of AI tools changes daily, so organizations must ride that wave and adapt continuously.
