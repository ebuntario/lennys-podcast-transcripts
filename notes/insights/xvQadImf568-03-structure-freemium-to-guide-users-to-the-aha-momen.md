In a freemium model, the core utility that gets a user to the "aha moment" and drives [[concepts/habit-formation|habit formation]] must be free. The goal is to shepherd users into a paid version by collapsing the time to value in the free tier, while strategically placing the paywall around advanced features that deliver value later in the user journey.
