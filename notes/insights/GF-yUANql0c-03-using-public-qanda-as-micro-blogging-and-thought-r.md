Answering founder questions via DMs serves as a form of [[concepts/thought-leadership|micro-blogging]] that bypasses writer's block by using external prompts. This practice helps refine one's own thinking on various topics while providing value, and sharing the anonymized exchanges publicly can help others with similar questions.
