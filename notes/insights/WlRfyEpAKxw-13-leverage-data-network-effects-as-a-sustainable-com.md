Data network effects, where product usage data makes the core product value stronger over time, are an underrated growth lever. This is not about selling data, but using it to improve outcomes like personalized recommendations (Pinterest) or better advertising targeting (Eventbrite). In a landscape where platform data (e.g., from Facebook) is becoming less accessible, building your own [[concepts/data-network-effects|data-driven advantages]] is a critical and sustainable edge.
