Before using AI to automate a process or build a tool, first ask if the activity is necessary at all. AI excels at execution but lacks global portfolio judgment. Humans must apply [[concepts/critical-thinking|critical thinking]] to question base assumptions, as sometimes changing the underlying process or deciding not to build something is more valuable than any automation.
