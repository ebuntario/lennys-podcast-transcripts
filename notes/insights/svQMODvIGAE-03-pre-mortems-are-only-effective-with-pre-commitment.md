A pre-mortem is great only if you attach a pre-commitment with it. Use the exercise to set up specific kill criteria—signals that things are going wrong—and commit to actions you will take if you see those signals. This creates structure to act rationally later, combating biases like [[concepts/sunk-cost-fallacy|sunk cost]].
