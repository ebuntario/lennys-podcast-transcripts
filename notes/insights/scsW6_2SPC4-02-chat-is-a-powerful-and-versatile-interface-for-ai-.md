Chat is an amazing interface for AI because it's the universal, unstructured way humans communicate. Unlike rigid interfaces, chat allows for maximum communication bandwidth, handling the full complexity and nuance of human language. This makes it a perfect fit for the power of [[concepts/ai-interfaces|large language models]], serving as a catch-all for any instruction you'd want to give.
