Simplify pricing by categorizing features as "day one" or "day 100" premium features. Day one features deliver immediate value and can be included in lower tiers to drive conversion. Day 100 features represent advanced functionality users appreciate later; placing these in higher tiers creates a clear [[concepts/upsell-strategy|upsell path]] as user sophistication and needs grow.
