Modern AI models are largely trained on open source code because it's one of the safest and most legally permissible corpora available. The licenses of open source explicitly allow this use, creating a virtuous cycle where open contributions fuel AI advancement. This ensures that [[concepts/open-source-ethics|openly licensed work]] becomes a core ingredient in the next generation of technological tools.
